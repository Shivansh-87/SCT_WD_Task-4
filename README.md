# Modern To-Do App

A clean, responsive, and accessible web-based to-do application that lets users add and maintain tasks organized in multiple lists. Users can mark tasks completed, edit tasks, and set date & time for tasks.

---

## Features

- **Multiple Lists:** Create and organize tasks into customizable lists.
- **Add Tasks:** Easily add tasks with optional due date and time.
- **Edit Tasks:** Inline editing for task names and due date/time.
- **Mark Completed:** Toggle tasks as completed with visual indication.
- **Delete Tasks:** Remove tasks individually.
- **Responsive Design:** Works smoothly on mobile, tablet, and desktop.
- **Accessible:** Keyboard navigable and screen reader friendly.
- **Modern UI:** Clean layout with material design icons and intuitive controls.

---

## How to Use

1. Open `index.html` in any modern web browser.
2. Use the sidebar to view existing lists or add a new list using the **+** button.
3. Click a list to view and manage its tasks.
4. Add tasks using the form at the top, including name and optional due date/time.
5. Mark tasks completed by checking their checkbox.
6. Edit a task by clicking the edit icon, change text or date/time, then press Enter or click outside.
7. Delete a task with the trash icon.
8. New lists and tasks update immediately without page reloads.

---
